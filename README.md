# barebone-restaurant-backen
Laravel backend for restaurants build for restaurants to scale .

## Features 
- [] Admin Panel for the Restaurant Owner
- [] CRUD for Products that can be shown on the Frontend.
- [] Product Types ex Breakfast Drinks and so on.
- [] Posibility of managing Notification for customers.
- [] Restaurant customers can create an account to give Reviews that will be shown on the Website.
- 


## How to use 

This backend barebone is just a starting point with some small features already implemented for restaurants backend . the Frontend is open for the user to implement ehat he wants . The admin panel is build with Filamentphp . But the customer frontend can be implented as wanted.

Just fork or clone this repo and get started with you`re project .
